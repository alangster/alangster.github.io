Responsive Design and Fluid Layouts

An introductory website to responsive design and fluid layouts.

By: Amber Hager and Alex Lang

What is Responsive Design?

Responsive design involves considering the site's structure and how objects fit into that structure. The focus is on creating a positive user experience, regardless of the medium through which the user views the page.1

Fluid Layouts

With fluid layouts, dimensions are set to percentages rather than pixels. Fluid layouts are more user friendly and have a simple design. Fluid layouts are linked to media queries and special styles for optomization.2

Conclusion

The purpose of fluid layouts is to allow good user experiences across a range of viewing devices (e.g., iPhone, tablet, computer, etc.) We think that it is good practice to use fluid layouts, because they make sites more accessible, allowing the information to reach as wide an audience as possible.

It is more complicated now because designers have to consider the different screens on which their pages may be viewed. At the same time, design has become more important because people are doing more and more (e.g., shopping) from mobile devices. In the future, I think wearable tech will change web design. Sites will need their designs to be formatted for devices like Google Glass, and such devices will also influence the way people interact with sites.

We should definitely consider visually impaired people when designing, but perhaps do so by offering a different version of the page for them because focusing solely on visually imparied viewers would limit what designers can do.
